# FruitSupermarket
Django REST framework + vue <br />
此项目是学习网上一个项目的代码所用的，学习Django REST framework + vue开发。<br />

## 环境
python: 3.6.4<br /> 
Django: 2.0.2
### 技术
* Vue + Django REST Framework前后端分离技术
* restful api开发流程
* Django REST Framework的功能实现和核心源码分析
* Sentry完成线上系统的错误日志的监控和告警
* 第三方登录和支付宝支付的集成
